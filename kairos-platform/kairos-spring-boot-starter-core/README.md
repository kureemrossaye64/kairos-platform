# kairos-spring-boot-starter-core

This starter provides the minimal foundation for any KAIROS-based application.

## Overview

The `kairos-spring-boot-starter-core` bundles the core domain models and auto-configuration logic. It is required for any application that wants to leverage the Kairos platform.

## Included Modules
- `kairos-core`
- `kairos-autoconfigure`

## Usage

Add the following dependency to your `pom.xml`:

```xml
<dependency>
    <groupId>com.kairos</groupId>
    <artifactId>kairos-spring-boot-starter-core</artifactId>
    <version>${project.version}</version>
</dependency>
```
