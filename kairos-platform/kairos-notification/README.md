# kairos-notification

The `kairos-notification` module provides a simple and extensible service for sending system-wide notifications.

## Overview

This module handles the delivery of alerts, status updates, and other messages to users or internal systems.

## Key Features

- **Centralized Notification Service**: A unified API for sending notifications.
- **Extensible Channels**: Easily add new notification channels (e.g., Email, SMS, Webhooks).

## Usage

```xml
<dependency>
    <groupId>com.kairos</groupId>
    <artifactId>kairos-notification</artifactId>
    <version>${project.version}</version>
</dependency>
```
