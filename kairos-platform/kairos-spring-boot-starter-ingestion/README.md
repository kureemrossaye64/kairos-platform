# kairos-spring-boot-starter-ingestion

This starter provides the RAG ingestion pipeline for your KAIROS application.

## Overview

The `kairos-spring-boot-starter-ingestion` bundles the ingestion services, document processors, and core platform features.

## Included Modules
- `kairos-ingestion`
- `kairos-spring-boot-starter-core`

## Usage

Add the following dependency to your `pom.xml`:

```xml
<dependency>
    <groupId>com.kairos</groupId>
    <artifactId>kairos-spring-boot-starter-ingestion</artifactId>
    <version>${project.version}</version>
</dependency>
```

Refer to the [kairos-ingestion](../kairos-ingestion) documentation for pipeline details.
