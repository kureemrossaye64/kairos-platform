# kairos-spring-boot-starter-crawler

This starter provides web crawling capabilities for your KAIROS application.

## Overview

The `kairos-spring-boot-starter-crawler` bundles the crawler service and core platform features.

## Included Modules
- `kairos-crawler`
- `kairos-spring-boot-starter-core`

## Usage

Add the following dependency to your `pom.xml`:

```xml
<dependency>
    <groupId>com.kairos</groupId>
    <artifactId>kairos-spring-boot-starter-crawler</artifactId>
    <version>${project.version}</version>
</dependency>
```

Refer to the [kairos-crawler](../kairos-crawler) documentation for configuration details.
