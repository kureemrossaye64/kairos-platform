# kairos-spring-boot-starter-agent

This starter provides everything needed to build and run AI agents using the KAIROS Agentic Framework.

## Overview

The `kairos-spring-boot-starter-agent` bundles the agentic framework, AI abstractions, and core platform features.

## Included Modules
- `kairos-agentic-framework`
- `kairos-ai-abstraction`
- `kairos-spring-boot-starter-core`

## Usage

Add the following dependency to your `pom.xml`:

```xml
<dependency>
    <groupId>com.kairos</groupId>
    <artifactId>kairos-spring-boot-starter-agent</artifactId>
    <version>${project.version}</version>
</dependency>
```

Refer to the [kairos-agentic-framework](../kairos-agentic-framework) documentation for more details on building agents.
