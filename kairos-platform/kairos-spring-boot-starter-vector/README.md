# kairos-spring-boot-starter-vector

This starter provides vector search and retrieval capabilities for your KAIROS application.

## Overview

The `kairos-spring-boot-starter-vector` bundles the vector search implementation and core platform features.

## Included Modules
- `kairos-vector-search`
- `kairos-spring-boot-starter-core`

## Usage

Add the following dependency to your `pom.xml`:

```xml
<dependency>
    <groupId>com.kairos</groupId>
    <artifactId>kairos-spring-boot-starter-vector</artifactId>
    <version>${project.version}</version>
</dependency>
```

Refer to the [kairos-vector-search](../kairos-vector-search) documentation for configuration details.
